現在のところ、Android エミュレータの実行は CircleCI 2.0 でサポートされていません。CircleCI ジョブからエミュレータのテストを実行するには、 Firebase Test Lab などの外部サービスをお使いください。

この操作は、gcloud Command Line を使用して行なえます (詳しいセットアップ方法などについては、ドキュメントを作成中です。 セットアップで現在お困りの場合は、Discuss でお知らせください)。gcloud ツールは、Android 用の CircleCI Docker イメージにプリインストールされています。詳しくは、Dockerfiles と Docker Hub を参照してください。
