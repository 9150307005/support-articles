Workflows を搭載した CircleCI 2.0 では、夜間ビルドをスケジュール できます。

CircleCI 1.0、または Worksflows なしの CircleCI 2.0 を使用している場合、CircleCI の API を使用して夜間ビルドをトリガーできます。

どちらを使用すべきかの選択

Workflows を搭載した CircleCI 2.0 を使用すると、CircleCI がお客様の代わりにビルドをスケジュールし、トリガーできます。 CircleCI の API を使用する場合、夜間のビルドをトリガーするため、お使いのサーバーの 1 つに cron をセットアップする必要があります。
